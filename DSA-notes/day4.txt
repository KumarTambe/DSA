ARRAYS — Day 4 patterns

1. Always initialise max/min to first element, not 0
   Reason: breaks on all-negative arrays

2. Running sum (prefix sum):
   Each element = itself + previous element
   Loop to size-1 when modifying arr[i+1]

3. Buy & Sell Stock pattern:
   Track minPrice and maxProfit in single pass
   At each element: update min, then check profit
   Time: O(n), Space: O(1)

4. Contains Duplicate:
   Use unordered_set — O(1) insert and lookup
   Insert and check existence in same loop
   Time: O(n), Space: O(n)
